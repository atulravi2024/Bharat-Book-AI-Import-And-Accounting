Bharat Book AI - Release Notes
Version 2.4.0 (Build 2404.19)
Released: May 2026

Bharat Book AI 2.4.0 represents our most stable and performant release yet, loaded with deep analytical enhancements, compliance updates, and critical system optimizations.

KEY ADVANCEMENTS & NEW FEATURES
======================================================
1. Intelligent Voucher Summary & Synthesis
   * Embedded state-of-the-art diagnostic summaries directly within the bank voucher and ledger tables.
   * Real-time automated cross-referencing checks for transaction mismatch detection.

2. Enhanced GST Compliance Engines
   * Upgraded high-speed validation for B2B transactions with proper Indian state code matching.
   * Robust mapping for B2C Large, B2C Small, Exports, and Tax-Exempt transactions directly from narrations.
   * Fine-tuned IGST vs. CGST/SGST automatic placement calculations based on State of Supply configurations.

3. High-Fidelity Ledger Synchronization
   * Improved support-grade ticket telemetry and offline security architecture.
   * Form layout matching for high-contrast viewing of financial ledgers.
   * Resolved minor visual clipping and icon rendering anomalies across both light and dark display contexts.

FIXES & STABILITY ENHANCEMENTS
======================================================
* Type-safe local notifications across Support and Directory management suites.
* Corrected layout bounds mapping for multi-device viewport transitions.
* Stabilized sample-dataset injection flow under isolated system tests.
